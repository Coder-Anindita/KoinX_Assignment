# KoinX Assignment 🚀

A React-based web application built using **Vite, React, Tailwind CSS, and Material UI** for crypto portfolio visualization and tax-related insights.

---

## 🛠️ Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Material UI (MUI)
- Emotion (styling engine for MUI)
- React Router DOM
- @Bootstrap

---

## 📦 Installation & Setup

### 1. Clone the repository
```bash
https://github.com/Coder-Anindita/koinx.git
```
### 2. Run on local server
```bash
cd koinX_Assignment
npm run dev
```

### 3. Project will be like on URL:
```bash
http://localhost:5173
```
